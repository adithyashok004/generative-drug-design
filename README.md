# Generative Drug Design with Drug-Likeness and Toxicity Prediction

An AI-powered system that generates novel drug molecules using Generative Deep Learning and evaluates them for drug-likeness, molecular properties, and toxicity to accelerate early-stage drug discovery.

---

## Overview

Drug discovery is a lengthy and expensive process. This project applies Generative AI to generate novel molecular structures and evaluates their pharmaceutical potential using computational chemistry techniques. The workflow combines molecular generation, drug-likeness assessment, toxicity prediction, and optimization into a single AI-driven pipeline.

---

## Features

- De novo drug molecule generation
- SMILES-based molecular representation
- Drug-likeness evaluation using Lipinski's Rule of Five
- Quantitative Estimate of Drug-likeness (QED)
- Molecular property prediction
- Toxicity prediction
- Hybrid quantum-classical optimization
- End-to-end AI-driven drug discovery workflow
